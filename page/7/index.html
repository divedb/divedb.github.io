Monte Carlo Simulation

1. 概念

​ 蒙特卡洛方法是一种基于随机抽样的计算方法，其主要思想是通过随机模拟大量样本来近似求解问题，通常用于以下几种情况：

  1. 概率估计：通过多次随机试验来估计事件发生的概率。例如，使用随机点落在几何图形内外来估计面积。
  2. 积分计算：在高维空间中，蒙特卡洛方法可以用于计算复杂积分，尤其是在解析解难以获得时。
  3. 优化问题：通过随机搜索空间来寻找函数的最优解，比如在机器学习中的超参数优化。
  4. 物理模拟：用于模拟粒子在物理系统中的行为，比如核反应或流体动力学。
Inferential Statistics

​ 推断统计（Inferential Statistics）是一种统计分析方法，用于从样本数据推断或估计总体特征。它通过对样本进行分析，来得出关于更大群体的结论或判断。推断统计的主要目的包括：

  1. 估计参数：通过样本数据估计总体参数（如均值、比例等），并提供置信区间来表示估计的不确定性。
  2. 假设检验：检验关于总体参数的假设，判断样本数据是否提供足够的证据支持或反驳某个假设。
  3. 样本间的比较：比较不同样本之间的特征差异，例如使用t检验或方差分析（ANOVA）。

推断统计依赖于概率论，使用概率模型来处理样本数据的不确定性，从而为决策提供依据。

Difference in Confidence?
  • Confidence in our estimate depends upon two things
  • Size of sample (100 versus 2)
  • Variance of sample (all heads versus 52 heads)
  • As the variance grows, we need larger samples to have the same degree of confidence
伯努利定理

伯努利定理可以表述为：

  1. 如果进行独立的伯努利试验，成功的概率为$p$，失败的概率为$1-p$，那么在 nnn 次试验中，成功的次数$X$服从二项分布$X \sim B(n, p)$。
  2. 在大量试验中，随着试验次数$n$的增加，样本比例趋向于真实概率$p$。
置信区间

​ 置信区间（Confidence Interval）是统计学中用于估计总体参数范围的一种方法。它提供了一个区间估计，表明在一定的置信水平下，该区间内可能包含未知的总体参数值。

  1. 置信水平：通常以百分比表示（如95%或99%），表示在重复抽样中，有多少比例的置信区间会包含真实的总体参数。例如，95%的置信水平意味着如果进行多次独立抽样，约95%的计算出的置信区间会包含真实的参数。

  2. 区间的计算：置信区间的计算通常依赖于样本均值、样本标准差和样本大小。常用的形式为：

    $\text{置信区间} = \text{样本均值} \pm z \times \frac{\text{样本标准差}}{\sqrt{n}}$

    其中，$z$是与置信水平对应的标准正态分布的临界值，$n$是样本大小。

  3. 解释：置信区间并不表示真实参数值在该区间内的概率，而是指在很多次抽样中，构造的置信区间有一定的比例会包含真实参数。

例题：

假设你想了解某个学校学生的数学考试成绩，随机抽取了30名学生，计算得出平均分为75分，标准差为10分。你希望构建一个95%的置信区间来估计该学校所有学生的平均分。

  1. 计算标准误差（SE）：

    $SE = \frac{\sigma}{\sqrt{n}} = \frac{10}{\sqrt{30}} \approx 1.83$

  2. 确定置信水平： 对于95%的置信水平，查找Z值（通常为1.96）。

  3. 计算置信区间：

    $\text{置信区间} = \bar{x} \pm Z \times SE = 75 \pm 1.96 \times 1.83$

因此，你可以说，你有95%的把握认为该学校所有学生的数学考试成绩平均分在71.41到78.59之间。

正态分布的概率密度函数

$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}}$

其中：

  • $\mu$是均值（期望值），表示正态分布的中心。
  • $\sigma$是标准差，表示数据的离散程度。
  • $e$是自然常数。
经验法则

经验法则（Empirical Rule）是一种用于描述正态分布数据集中数据点分布的统计规则。根据这一规则，约有：

  1. 68% 的数据点位于均值的一个标准差范围内（即$\mu \pm \sigma$）。
  2. 95% 的数据点位于均值的两个标准差范围内（即$\mu \pm 2\sigma$）。
  3. 99.7% 的数据点位于均值的三个标准差范围内（即$\mu \pm 3\sigma$）。

Introduction to Reinforcement Learning

1. 概念

​ 强化学习（Reinforcement Learning, RL）是一种机器学习的范式，旨在通过与环境的交互来学习如何采取最佳行动，以最大化累积的奖励。其核心思想是通过试错的方式，让智能体在给定的环境中探索和学习。

强化学习的基本要素包括：

  1. 智能体（Agent）: 在环境中做出决策的实体。
  2. 环境（Environment）: 智能体与之交互的外部系统，包括状态、奖励和状态转移。
  3. 状态（State）: 环境在某一时刻的描述，智能体根据状态来做出决策。
  4. 动作（Action）: 智能体在某个状态下可以选择的操作。
  5. 奖励（Reward）: 智能体在执行某个动作后，环境给予的反馈信号，用于评估该动作的好坏。
  6. 策略（Policy）: 智能体选择动作的规则，可以是确定性的（给定状态下选择固定动作）或随机的（根据概率分布选择动作）。
  7. 价值函数（Value Function）: 用于评估在某个状态下的期望收益，帮助智能体判断哪个状态更有利。

2. State

  • Environment state

  • Agent state

  • Markov state

    • 马尔可夫性：当前状态的未来状态只依赖于当前状态，而与过去的状态无关。换句话说，给定当前状态，未来的状态转移不受过去历史的影响。

    • 在强化学习或随机过程的背景下，马尔可夫状态通常指的是系统的状态，该状态能够完整描述系统的未来演变。也就是说，知道当前状态后，未来状态的概率分布只与当前状态相关。

    例子

    • 在棋类游戏中，当前的棋盘布局就是一个马尔可夫状态，因为下一步的可能棋盘布局仅依赖于当前的布局，而与之前的动作和状态无关。
    • 在天气预报中，如果当前状态是“晴天”，那么明天的天气只依赖于“晴天”的状态，而与之前的天气无关。

3. Observability

完全可观测性（Full Observability）
  • 定义：在完全可观测的系统中，所有的状态信息都可以被完全观测到。即，在每个时间点，系统的状态是完全可知的。
  • 特点
    • 代理可以直接获得系统的当前状态信息。
    • 不需要任何额外的推理或估计。
  • 例子：在一个简单的棋类游戏中，玩家可以看到整个棋盘的状态，这就是一个完全可观测的环境。
部分可观测性（Partial Observability）
  • 定义：在部分可观测的系统中，代理只能观察到系统状态的一部分信息，或可能存在噪声，使得状态无法完全确定。
  • 特点
    • 代理必须依赖于观察到的信息来推测未观察到的状态。
    • 需要使用策略来处理不确定性，通常涉及状态估计或历史信息。
  • 例子：在机器人导航中，机器人可能只能感知周围的障碍物，而无法知道环境的完整布局。
应用
  • 在强化学习中，部分可观测环境通常需要使用部分可观测马尔可夫决策过程（POMDP）模型来处理，而完全可观测环境则可以简单地使用马尔可夫决策过程（MDP）模型。

4. Major Components of an RL Agent

  • An RL agent may include one or more of these components:

    • Policy: agent’s behaviour function

    • Value function: how good is each state and/or action

      价值函数（Value Function）是强化学习中的一个核心概念，用于评估某一状态或状态-动作对的“价值”。它帮助代理（agent）决定在不同状态下采取哪些行动，以最大化长期回报。

      主要类型

      1. 状态价值函数（State Value Function, V(s)）：
        • 定义：在给定状态$s$下，代理在该状态下所能获得的预期回报。
        • 表示为：$V(s) = \mathbb{E}[R_t | S_t = s]$，其中$R_t$是从时间$t$开始的累计回报。
      2. 动作价值函数（Action Value Function, Q(s, a)）：
        • 定义：在给定状态$s$和动作$a$下，代理执行该动作后所能获得的预期回报。
        • 表示为：$Q(s, a) = \mathbb{E}[R_t | S_t = s, A_t = a]$。

      在象棋中，假设当前状态是棋盘上某种特定的布局：

      • 状态价值函数（V(s)）：如果在当前布局下，玩家有很高的胜算，比如能确保胜利或导致对手陷入绝境，状态价值函数$V(s)$可以被认为是正值，比如+100，表示这是一个有利的状态。
      • 动作价值函数（Q(s, a)）：假设在该状态下，玩家选择了一步棋（比如移动某个棋子）但这一步棋导致了对手的反击，最终失败。此时，动作价值函数$Q(s, a)$可能是负值，比如-50，表示这一步棋在这个状态下是个不好的选择。

    • Model: agent’s representation of the environment

FAQ

1. state value function $V(s)$ and state-action value function $Q(s, a)$ difference?

​ Bearing that in mind, the definition of these functions should clarify the distinction for you.

$V_{\pi}(s)$ expresses the expected value of following policy $\pi$ forever when the agent starts following it from state $s$.

$Q_{\pi}(s, a)$ expresses the expected value of first taking action $a$ from state $s$ and then following policy $\pi$ forever.

The main difference then, is the Q-value lets you play a hypothetical of potentially taking a different action in the first time step than what the policy might prescribe and then following the policy from the state the agent winds up in.

Example:

That said, lets make the example more concrete. Lets imagine a deterministic grid world. The goal is at (5, 0). As stated in the description, the agent is one step away, so lets say s = (4, 0) for concreteness. For the optimal policy, it will tell us to go right from (4, 0). If we take right, that takes us to our terminating goal state (5, 0), with a single reward of -1 for taking that action after which we stop and receive no more reward. Clearly then, $V^*((4, 0)) = -1$.

As stated in the answer, we want to compare this to the Q-value for an action that takes us one step further away than where we started. Let say $a$ = left.

If we start at (4, 0) and take left, then we’ll end up in (3, 0), with -1 reward.

From here on out we now follow our policy. The agent is now in state (3, 0) and our policy says to take right. That takes us back to where we started: (4, 0) receiving another reward of -1, for a total of -2 reward so far.

But we’re still not done, because we’re only back to where we started! Once again, we follow our policy which from (4, 0) tells us to go right again, now taking us to our terminating goal state (5, 0) with another -1 reward for a total of -3 reward.

So we just accumulated -3 reward; ergo, $Q^*((4, 0), left) = -3$.

偏微分概念

1. Gradient（梯度）

​ Gradient是向量微积分中的一个重要概念，它描述了多变量函数的导数。梯度是一个向量，表示函数在各个自变量方向上的变化率。具体来说，它是多变量函数的偏导数的集合。

解释：

对于一个多元函数$f(x_1, x_2, \dots, x_n)$，梯度$\nabla f$是一个向量，由函数对每个变量的偏导数组成：

$\nabla f(x_1, x_2, \dots, x_n) = \left( \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n} \right)$

每个分量$\frac{\partial f}{\partial x_i}$表示函数$f$在变量$x_i$方向上的变化率（即对该变量的导数）。

举例：

假设$f(x, y) = x^2 + y^2$，则它的梯度为：

$\nabla f(x, y) = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right) = \left( 2x, 2y \right)$

这意味着函数$f$在$x$方向的变化率是$2x$，在$y$方向的变化率是$2y$。

应用：

  • 机器学习中的梯度下降： 在机器学习中的优化问题（如神经网络训练）中，梯度用于找到损失函数的最小值。通过计算损失函数的梯度，模型参数会沿着负梯度方向更新，逐步逼近最优解。这就是梯度下降法。
梯度下降算法步骤

  1. 初始化参数：随机或以某种方式初始化模型的参数（如权重和偏置）。

  2. 计算损失：将输入数据通过模型进行预测，计算损失函数的值，表示模型预测和实际目标之间的差异。

  3. 计算梯度：对每一个参数，计算损失函数相对于该参数的梯度，表示在当前参数值下，如何调整参数能使损失减小。

  4. 更新参数：通过公式：

    $\theta = \theta - \eta \nabla J(\theta)$

    其中，$\theta$是参数，$\eta$是学习率，$\nabla J(\theta)$是损失函数对参数的梯度。学习率控制每次更新的步长大小。

  5. 重复迭代：重复步骤 2 到 4，直到损失函数收敛（达到最小值）或者满足其他终止条件（如达到设定的迭代次数）。

梯度下降的类型

梯度下降有几种常见的变体：

  1. 批量梯度下降（Batch Gradient Descent）：
    • 每次更新参数时，使用整个训练集来计算梯度。
    • 优点：更新方向更加稳定。
    • 缺点：对于大数据集，计算梯度耗时。
  2. 随机梯度下降（Stochastic Gradient Descent，SGD）：
    • 每次更新参数时，使用一个训练样本来计算梯度。
    • 优点：计算更快，适合大数据集。
    • 缺点：更新方向可能不稳定，收敛过程中存在波动。
  3. 小批量梯度下降（Mini-Batch Gradient Descent）：
    • 每次更新参数时，使用一个小的训练样本子集来计算梯度。
    • 结合了批量梯度下降和随机梯度下降的优点，既提高了计算效率，又保持了更新的稳定性。
学习率的重要性
  • 学习率过大：可能导致更新步长太大，跳过了最优解，无法收敛。
  • 学习率过小：更新步长太小，训练过程非常慢，可能会陷入局部最小值。

通常在实际使用中，选择一个合适的学习率非常重要，并且有时会采用学习率衰减的策略，即随着训练过程的进行逐渐减小学习率。

变种算法

梯度下降算法有很多变种，比如：

  • Momentum：通过引入动量来加速梯度下降。
  • Adam（Adaptive Moment Estimation）：结合了动量和自适应学习率，表现优秀且广泛使用。
应用

​ 梯度下降用于深度学习模型的训练，尤其是神经网络的权重更新。通过反向传播算法，梯度被逐层计算，从输出层逐渐传递到输入层，使得网络的所有层都能通过梯度下降调整其参数。

2. 反向传播

​ 反向传播（Backpropagation） 是一种用于训练人工神经网络的算法。它的主要目的是通过计算损失函数相对于每个权重的梯度，以更新网络的权重，从而最小化预测误差。以下是反向传播的主要步骤和概念：

1. 前向传播（Forward Pass）
  • 输入数据通过网络层层传递，每一层都应用激活函数。最终产生模型的输出（预测值）。
  • 计算损失函数，通常是输出值与真实标签之间的差异（如均方误差、交叉熵等）。
2. 反向传播（Backward Pass）

  • 计算梯度：从输出层开始，计算损失函数对每个输出的梯度。然后，利用链式法则逐层向后计算每个神经元的梯度。这涉及到以下几个步骤：

    • 计算输出层的梯度。
    • 对于每一层，利用当前层的梯度和激活函数的导数，计算前一层的梯度。

  • 权重更新：使用计算出的梯度更新网络中的权重。常用的优化算法是梯度下降法（Gradient Descent），更新公式通常为：

    $w = w - \eta \cdot \frac{\partial L}{\partial w}$

    其中$w$是权重，$\eta$是学习率，$L$是损失函数。

3. 重复训练
  • 通过多次前向传播和反向传播循环（通常称为“epochs”），模型不断调整权重，以提高准确性并减少损失。
反向传播的关键点
  • 链式法则：反向传播依赖于链式法则，用于计算复合函数的导数。
  • 计算效率：反向传播算法通过逐层传播梯度，显著减少了计算量，提高了训练效率。
  • 适用于深层网络：反向传播能够处理具有多个隐藏层的深层神经网络。

Probability Measure

1. 概念

​ Probability measure is a mathematical concept used in probability theory to assign a numerical value to the likelihood of certain events occurring within a given sample space. It formalizes the idea of measuring “how likely” an event is.

Key components of a probability measure:
  1. Sample Space (Ω): The set of all possible outcomes of a random experiment. For example, when flipping a coin, the sample space is ${\text{Heads}, \text{Tails} }$.
  2. Events: Subsets of the sample space that we are interested in. An event can be a single outcome or a group of outcomes. For example, in rolling a die, the event of getting an even number is ${2, 4, 6}$.
  3. Probability Measure (P): A function that assigns a probability (a number between 0 and 1) to each event in the sample space. It satisfies three axioms (the Kolmogorov axioms):
    • Non-negativity: For any event $A, P(A) \geq 0$.
    • Normalization: $P(\Omega) = 1$, meaning that the probability of the entire sample space is 1.
    • Additivity: If $A_1, A_2, \dots$ are disjoint events (mutually exclusive), then $P(A_1 \cup A_2 \cup \dots) = P(A_1) + P(A_2) + \dots$

例题：

​ In a fair six-sided die roll, the sample space $\Omega = {1, 2, 3, 4, 5, 6}$. A probability measure could be defined as:

  • $P({1}) = \frac{1}{6}, P({2}) = \frac{1}{6}, \dots, P({6}) = \frac{1}{6}$. In this case, each outcome has an equal probability of occurring, and the sum of the probabilities over all possible outcomes is 1.

2. Sigma-algebra的定义：

​ 假设$\Omega$是一个非空集合（通常称为样本空间），则一个 $\sigma$代数$\mathcal{F}$是该集合上的一个子集族，满足以下三个条件：

  1. 包含全集：$\Omega \in \mathcal{F}$。即整个样本空间是$\sigma$代数中的一个元素。
  2. 闭合于补集：如果$A \in \mathcal{F}$，则$A^c \in \mathcal{F}$，其中$A^c$表示事件$A$的补集。也就是说，$\sigma$代数中的每个事件的补集也必须在$\sigma$代数中。
  3. 闭合于可列并：如果$A_1, A_2, A_3, \dots \in \mathcal{F}$ 是$\sigma$代数中的一列事件，那么它们的并集$\bigcup_{n=1}^{\infty} A_n \in \mathcal{F}$也必须在$\sigma$代数中。这意味着$\sigma$代数对于可数的并集运算是封闭的。
解释：
  • 事件：在概率论中，事件可以看作是样本空间$\Omega$的子集。$\sigma$代数就是一组我们可以分配概率的事件集。
  • 闭合性：$\sigma$代数必须对补集操作和无限并集操作封闭，这样可以确保定义在这个代数上的测度（例如概率测度）有良好的性质，并且我们可以合理地处理复杂的事件。
示例：
  1. 最小的$\sigma$代数：最简单的情况是$\Omega = {1, 2, 3}$，最小的$\sigma$代数就是$\mathcal{F} = {\emptyset, \Omega}$，仅包含空集和全集。
  2. 幂集：样本空间$\Omega = {1, 2}$，它的幂集（即所有子集的集合）就是$\mathcal{F} = {\emptyset, {1}, {2}, {1, 2}}$。幂集是一个$\sigma$代数，因为它满足上述三条条件。
  3. Borel $\sigma$代数：在实数集合$\mathbb{R}$上，最常见的$\sigma$代数是Borel $\sigma$代数，它由所有的开区间$(a,b) \subseteq \mathbb{R}$ 生成。Borel $\sigma$代数包含了实数集的所有可以通过开区间构造出来的集合，也是定义概率测度和Lebesgue测度的重要工具。
重要性：
  • 定义概率空间：在概率论中，样本空间$\Omega$上的$\sigma$代数$\mathcal{F}$和概率测度$P$一起构成了一个概率空间$(\Omega, \mathcal{F}, P)$，这允许我们对事件集分配概率。
  • 可测性：$\sigma$代数是可测空间的基础，随机变量的定义依赖于$\sigma$代数。随机变量是可测函数，即它将样本空间$\Omega$映射到另一个空间的过程中，保持可测性结构。

LU分解

1. 概念

​ LU分解（LU Decomposition）是一种将一个矩阵分解为两个特定矩阵的方法，通常用于解决线性方程组、计算行列式以及求逆矩阵。LU 分解的主要思想是将一个给定的方阵$A$分解为两个矩阵的乘积：一个下三角矩阵$L$和一个上三角矩阵$U$。

定义

如果$A$是一个$n \times n$的可逆矩阵，则可以表示为：
$$
A = LU
$$

2. 例题

设矩阵$A$为：
$$
A = \begin{pmatrix}
4 & 3 & -1 & 2 \
2 & 1 & 5 & 7 \
1 & 2 & 4 & -3 \
5 & 4 & 3 & 1
\end{pmatrix}
$$
我们希望找到$L$和$U$，使得$A=LU$。

首先初始化$U$为矩阵$A$，并将$L$初始化为单位矩阵：
$$
L = \begin{pmatrix}
1 & 0 & 0 & 0 \
0.5 & 1 & 0 & 0 \
0.25 & 0 & 1 & 0 \
1.25 & 0 & 0 & 1
\end{pmatrix}
$$

$$
U = \begin{pmatrix}
4 & 3 & -1 & 2 \
2 & 1 & 5 & 7 \
1 & 2 & 4 & -3 \
5 & 4 & 3 & 1
\end{pmatrix}
$$

对第1行对系数进行消元，将第2、3、4行的首列元素消为0。

  • $L_{21}=\frac{U_{21}}{U_{11}}=\frac{2}{4}=0.5$
  • $L_{21}=\frac{U_{31}}{U_{11}}=\frac{1}{4}=0.25$
  • $L_{41}=\frac{U_{41}}{U_{11}}=\frac{5}{4}=1.25$

使用这些系数对第2、3、4行进行变换，并更新$U$：

  • $U_2 \Rightarrow U_2 - 0.5 \times U_1$
  • $U_3 \Rightarrow U_3 - 0.25 \times U_1$
  • $U_4 \Rightarrow U_4 - 1.25 \times U_1$

计算得出：
$$
U = \begin{pmatrix}
4 & 3 & -1 & 2 \
0 & -0.5 & 5.5 & 6 \
0 & 1.25 & 4.25 & -3.5 \
0 & 0.25 & 4.25 & -1.5
\end{pmatrix}
$$
同时更新$L$：

接下来对第2列进行消元，使$U$中第3、4行的第二列元素为0：

  • $L_{32}=\frac{U_{32}}{U_{22}}=\frac{1.25}{-0.5}=-2.5$
  • $L_{42}=\frac{U_{42}}{U_{22}}=\frac{0.25}{-0.5}=-0.5$

进行行变换，更新$U$：

  • $U_3 \Rightarrow U_3 + 2.5 \times U_2$
  • $U_4 \Rightarrow U_4 + 0.5 \times U_2$

计算得出：
$$
U = \begin{pmatrix}
4 & 3 & -1 & 2 \
0 & -0.5 & 5.5 & 6 \
0 & 0 & 18 & 11.5 \
0 & 0 & 7 & 1.5
\end{pmatrix}
$$
同时更新$L$：
$$
L = \begin{pmatrix}
1 & 0 & 0 & 0 \
0.5 & 1 & 0 & 0 \
0.25 & -2.5 & 1 & 0 \
1.25 & -0.5 & 0 & 1
\end{pmatrix}
$$
最后，对第3列进行消元：

  • $L_{43}=\frac{U_{43}}{U_{33}}=\frac{7}{18} \approx 0.389$

进行行变换，更新$U$：

$U_4 \to U_4 - 0.389 \times U_3$

计算得出：
$$
U = \begin{pmatrix}
4 & 3 & -1 & 2 \
0 & -0.5 & 5.5 & 6 \
0 & 0 & 18 & 11.5 \
0 & 0 & 0 & -2.972
\end{pmatrix}
$$
同时更新$L$：
$$
L = \begin{pmatrix}
1 & 0 & 0 & 0 \
0.5 & 1 & 0 & 0 \
0.25 & -2.5 & 1 & 0 \
1.25 & -0.5 & 0.389 & 1
\end{pmatrix}
$$

海涅定理

1. 概念

​ 归结原则，又称为海涅（Heine）定理，即：设$f(x)$在$x_0$的某空心邻域内有定义，那么$f$在$x_0$处有极限$L$的充分必要条件是：对于任何一个收敛于$x_0$的数列${x_n \ne x_0: n = 1, 2, 3, \ldots}$，数列${f(x_n)}$有极限$L$。

证明必要性：

由$\lim_{x \to x_0}f(x)$存在，设$\lim_{x \to x_0}=A$，即$\forall \epsilon \gt 0, \exists \delta \gt 0$，当$0 \lt |x - x_0| \lt \delta$时，都有$|f(x) - A| \lt \epsilon, \forall {x_n} \in \overset{0}{U}(x_0), \lim_{n \to \infty}x_n = x_0$，要证明$\lim_{n \to \infty}f(x_n) = A$。

只需要证明$\forall \epsilon \gt 0, \exists N$，当$n \gt N$时，都有$|f(x_n) - A| \lt \epsilon$。由$\lim_{n \to \infty}x_n = x_0$，对上述$\delta \gt 0, \exists N$，当$n \gt N$时，都有$0 \lt |x_n - x_0| \lt \delta$，都有$|f(x_n) - A| \lt \epsilon$。

证明充分性：

由$\forall {x_n} \in \overset{0}U(x_0), \lim_{n \to \infty}x_n = x_0$，都有$\lim_{n \to \infty}(x_n)$存在且相等，设极限值为$A$。

我们要证$\lim_{x \to x_0}f(x) = A$。

用反证法：假设$f(x)$当$x \to x_0$时，不以$A$为极限，即$\exists \epsilon \gt 0$，对无论多么小的$\delta \gt 0, \exists x_\delta$。虽然$0 \lt |x_\delta - x_0| \lt \delta$但是$|f(x_\delta - A) \ge \epsilon_0$。

  • 取$\frac{\delta}{2} \gt 0, \exists x_1, 0 \lt |x_1 - x_0| \lt \frac{\delta}{2}$，但$|f(x_1) - A| \ge \epsilon_0$。
  • 取$\frac{\delta}{2^2} \gt 0, \exists x_2, 0 \lt |x_1 - x_0| \lt \frac{\delta}{2^2}$，但$|f(x_2) - A| \ge \epsilon_0$。
  • 取$\frac{\delta}{2^n} \gt 0, \exists x_n, 0 \lt |x_n - x_0| \lt \frac{\delta}{2^n}$，但$|f(x_n) - A| \ge \epsilon_0$。

构造出一个数列，${x_n} \in \overset{0}U(x_0, \delta), 0 \lt |x_n - x_0| \lt \frac{\delta}{2^n} \Rightarrow \lim_{n \to \infty}x_n = x_0$。

但是$|f(x_n) - A| \ge \epsilon$知$f(x_n)$当$n$趋于无穷大时不以$A$为极限矛盾。
