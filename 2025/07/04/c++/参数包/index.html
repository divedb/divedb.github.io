    参数包

    发表于 2025-07-04 分类于 C++

    在C++中，pack（参数包）是一种可接受多个元素的语言实体，包含以下几类：

    • 参数包（parameter pack）

      • 模板参数包（template parameter pack）
      • 函数参数包（function parameter pack）

    • Lambda初始化捕获包（lambda init-capture pack）(C++20起)

    • 结构化绑定包（structured binding pack）(C++26起)

    模板参数包是一种模板参数，它接受零个或多个模板实参（可以是常量、类型或模板）。函数参数包是一种函数参数，它接受零个或多个函数实参。

    Lambda初始化捕获包是一种Lambda捕获，它为其初始化器中包展开的每个元素引入一个初始化捕获。(C++20起)

    结构化绑定包是结构化绑定声明中的一个标识符，它引入零个或多个结构化绑定。(C++26起)

    一个包的元素数量等于：

    • 如果包是模板或函数参数包，则等于为其提供的实参数量。
    • 如果包是Lambda初始化捕获包，则等于其初始化器中包展开的元素数量。(C++20起)
    • 如果包是结构化绑定包，则等于初始化器的结构化绑定大小减去结构化绑定声明中非包元素的数量。(C++26起)

    至少包含一个参数包的模板被称为可变参数模板（variadic template）。

    语法

    模板参数包（出现在别名模板、类模板、变量模板（C++14起）、概念（C++20起）和函数模板的参数列表中）：

    1. type ... pack-name(可选)：一个带有可选名称的常量模板参数包。
    2. typename|class ... pack-name(可选)：一个带有可选名称的类型模板参数包。
    3. type-constraint ... pack-name(可选)：一个带有可选名称的受限类型模板参数包。(C++20起)
    4. template < parameter-list > class ... pack-name(可选) (C++17前)或template < parameter-list > typename|class ... pack-name(可选)(C++17起)：一个带有可选名称的模板模板参数包。

    函数参数包（一种声明符形式，出现在可变参数函数模板的函数参数列表中）：

    1. pack-name ... pack-param-name(可选)：一个带有可选名称的函数参数包。

    对于非参数包的语法，请参见Lambda初始化捕获包和结构化绑定包(C++26 起)。(C++20起)

    Here’s a refined translation and explanation:

    Pack (C++11 起)

    C++ 语言 | 声明 | 表达式 | 模板

    在 C++ 中，包（pack） 是一种实体，它定义了以下之一：

    • 参数包（parameter pack）
      • 模板参数包（template parameter pack）
      • 函数参数包（function parameter pack）
    • Lambda 初始化捕获包（lambda init-capture pack） (C++20 起)
    • 结构化绑定包（structured binding pack） (C++26 起)

    模板参数包是一种模板参数，它接受零个或多个模板实参（可以是常量、类型或模板）。函数参数包是一种函数参数，它接受零个或多个函数实参。

    Lambda 初始化捕获包是一种 Lambda 捕获，它为其初始化器中包展开的每个元素引入一个初始化捕获。(C++20 起)

    结构化绑定包是结构化绑定声明中的一个标识符，它引入零个或多个结构化绑定。(C++26 起)

    一个包的元素数量等于：

    • 如果包是模板或函数参数包，则等于为其提供的实参数量。
    • 如果包是 Lambda 初始化捕获包，则等于其初始化器中包展开的元素数量。(C++20 起)
    • 如果包是结构化绑定包，则等于初始化器的结构化绑定大小减去结构化绑定声明中非包元素的数量。(C++26 起)

    至少包含一个参数包的模板被称为可变参数模板（variadic template）。

    语法

    模板参数包（出现在别名模板、类模板、变量模板 (C++14 起)、概念 (C++20 起) 和函数模板的参数列表中）：

    1. type ... pack-name (可选)：一个带有可选名称的常量模板参数包。
    2. typename|class ... pack-name (可选)：一个带有可选名称的类型模板参数包。
    3. type-constraint ... pack-name (可选)：一个带有可选名称的受限类型模板参数包。(C++20 起)
    4. template < parameter-list > class ... pack-name (可选) (C++17 前) 或 template < parameter-list > typename|class ... pack-name (可选) (C++17 起)：一个带有可选名称的模板模板参数包。

    函数参数包（一种声明符形式，出现在可变参数函数模板的函数参数列表中）：

    1. pack-name ... pack-param-name (可选)：一个带有可选名称的函数参数包。

    对于非参数包的语法，请参见 Lambda 初始化捕获包 和 结构化绑定包 (C++26 起)。(C++20 起)

    展开的位置（Expansion Loci）

    参数包可用于多种语境：

    • 函数实参列表：f(args...)
    • 直接初始化：Class c(args...)
    • 花括号初始化：int a[] = {1, args..., 2};
    • 模板实参列表：container<A, B, C...>
    • 函数参数列表：void f(Ts... args)
    • 模板参数列表：template<T... Values>
    • 基类列表和成员初始化：class X : public Mixins... { X(const Mixins&... mixins) : Mixins(mixins)... {} };
    • Lambda 捕获：[args...]
    • 动态异常规范（C++17 前）：throw(X...)
    • alignas 对齐说明符：alignas(T...)
    • 属性列表：[[attr(args)...]]
    • using 声明：using bases::f...;
    • 折叠表达式（Fold expression）
    • 包索引（自 C++26）：args...[0] + args...[sizeof...(args) - 1]
    • friend 声明：friend Ts...;
