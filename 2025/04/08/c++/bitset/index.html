    C++ bitset 使用笔记

    bitset 是 C++ 标准库中一个高效的位操作容器，特别适合处理固定大小的位标志集合。以下是我的使用总结：

    一、基本特性

    1. 固定大小：大小在编译时确定，不可动态调整
    2. 高效存储：每个元素仅占1位物理空间
    3. 快速操作：支持各种位运算，直接映射到CPU指令
    4. 转换能力：可与整数、字符串相互转换

    六、注意事项

    1. 大小限制：to_ulong() 可能抛出 overflow_error
    2. 越界访问：使用 test() 会检查边界，operator[] 不会
    3. 非标准容器：没有迭代器，不能用于STL算法
    4. 平台差异：不同平台可能有不同的大小限制

    八、总结

    bitset 是处理固定大小位集合的最佳选择，它：

    • 比 vector<bool> 更高效
    • 比原始位操作更安全
    • 比整数位掩码更易读
    • 特别适合硬件寄存器操作、位标志集合、位图算法等场景

    合理使用 bitset 可以写出既高效又易维护的位操作代码。
