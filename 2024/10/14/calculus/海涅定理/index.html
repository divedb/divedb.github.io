海涅定理

1. 概念

​ 归结原则，又称为海涅（Heine）定理，即：设$f(x)$在$x_0$的某空心邻域内有定义，那么$f$在$x_0$处有极限$L$的充分必要条件是：对于任何一个收敛于$x_0$的数列${x_n \ne x_0: n = 1, 2, 3, \ldots}$，数列${f(x_n)}$有极限$L$。

证明必要性：

由$\lim_{x \to x_0}f(x)$存在，设$\lim_{x \to x_0}=A$，即$\forall \epsilon \gt 0, \exists \delta \gt 0$，当$0 \lt |x - x_0| \lt \delta$时，都有$|f(x) - A| \lt \epsilon, \forall {x_n} \in \overset{0}{U}(x_0), \lim_{n \to \infty}x_n = x_0$，要证明$\lim_{n \to \infty}f(x_n) = A$。

只需要证明$\forall \epsilon \gt 0, \exists N$，当$n \gt N$时，都有$|f(x_n) - A| \lt \epsilon$。由$\lim_{n \to \infty}x_n = x_0$，对上述$\delta \gt 0, \exists N$，当$n \gt N$时，都有$0 \lt |x_n - x_0| \lt \delta$，都有$|f(x_n) - A| \lt \epsilon$。

证明充分性：

由$\forall {x_n} \in \overset{0}U(x_0), \lim_{n \to \infty}x_n = x_0$，都有$\lim_{n \to \infty}(x_n)$存在且相等，设极限值为$A$。

我们要证$\lim_{x \to x_0}f(x) = A$。

用反证法：假设$f(x)$当$x \to x_0$时，不以$A$为极限，即$\exists \epsilon \gt 0$，对无论多么小的$\delta \gt 0, \exists x_\delta$。虽然$0 \lt |x_\delta - x_0| \lt \delta$但是$|f(x_\delta - A) \ge \epsilon_0$。

  • 取$\frac{\delta}{2} \gt 0, \exists x_1, 0 \lt |x_1 - x_0| \lt \frac{\delta}{2}$，但$|f(x_1) - A| \ge \epsilon_0$。
  • 取$\frac{\delta}{2^2} \gt 0, \exists x_2, 0 \lt |x_1 - x_0| \lt \frac{\delta}{2^2}$，但$|f(x_2) - A| \ge \epsilon_0$。
  • 取$\frac{\delta}{2^n} \gt 0, \exists x_n, 0 \lt |x_n - x_0| \lt \frac{\delta}{2^n}$，但$|f(x_n) - A| \ge \epsilon_0$。

构造出一个数列，${x_n} \in \overset{0}U(x_0, \delta), 0 \lt |x_n - x_0| \lt \frac{\delta}{2^n} \Rightarrow \lim_{n \to \infty}x_n = x_0$。

但是$|f(x_n) - A| \ge \epsilon$知$f(x_n)$当$n$趋于无穷大时不以$A$为极限矛盾。
