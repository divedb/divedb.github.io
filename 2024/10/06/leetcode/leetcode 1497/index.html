leetcode 1497

要求

Given an array of integers arr of even length n and an integer k.

We want to divide the array into exactly n / 2 pairs such that the sum of each pair is divisible by k.

Return true If you can find a way to do that or false otherwise.

Constraints:

  • arr.length == n

  • -109 <= arr[i] <= 109

思路

Let the array be $A=[a_1, a_2, … , a_n]$ and the divisor be k. We need to form pairs such that:

$(a_i+a_j)\mod k = 0$

This can be rewritten as:

$(a_i\mod k + a_j\mod k) \mod k=0$

For each element in the array, its remainder when divided by k lies in the range [0,k−1]. Let’s denote the remainder of an element $a_i$ by $mod_i = a_i\mod k$. To form a valid pair ($a_i$,$a_j$), we need:

$(mod_i + mod_j)\mod k = 0$

This implies:

$mod_i = k - mod_j$
