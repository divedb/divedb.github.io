Python Mixin

1. 简介

​ Mixin是一种设计模式，用于创建可复用的类，这些类可以与其他类组合，以增强其功能。Mixin通常不作为独立的类使用，而是作为基类被其他类继承，提供特定的行为或功能，而不需要成为该类的主要部分。

特点
  1. 可复用性：Mixin允许将功能模块化，使得相同的功能可以在多个类中复用。
  2. 单一职责：Mixin通常负责提供一种特定的功能或行为，而不是承担类的主要业务逻辑。这有助于保持代码的清晰性和可维护性。
  3. 组合性：可以将多个Mixin类组合到一个类中，从而为该类提供多种功能。这种方式支持多重继承。
