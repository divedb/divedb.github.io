Probability Measure

1. 概念

​ Probability measure is a mathematical concept used in probability theory to assign a numerical value to the likelihood of certain events occurring within a given sample space. It formalizes the idea of measuring “how likely” an event is.

Key components of a probability measure:
  1. Sample Space (Ω): The set of all possible outcomes of a random experiment. For example, when flipping a coin, the sample space is ${\text{Heads}, \text{Tails} }$.
  2. Events: Subsets of the sample space that we are interested in. An event can be a single outcome or a group of outcomes. For example, in rolling a die, the event of getting an even number is ${2, 4, 6}$.
  3. Probability Measure (P): A function that assigns a probability (a number between 0 and 1) to each event in the sample space. It satisfies three axioms (the Kolmogorov axioms):
    • Non-negativity: For any event $A, P(A) \geq 0$.
    • Normalization: $P(\Omega) = 1$, meaning that the probability of the entire sample space is 1.
    • Additivity: If $A_1, A_2, \dots$ are disjoint events (mutually exclusive), then $P(A_1 \cup A_2 \cup \dots) = P(A_1) + P(A_2) + \dots$

例题：

​ In a fair six-sided die roll, the sample space $\Omega = {1, 2, 3, 4, 5, 6}$. A probability measure could be defined as:

  • $P({1}) = \frac{1}{6}, P({2}) = \frac{1}{6}, \dots, P({6}) = \frac{1}{6}$. In this case, each outcome has an equal probability of occurring, and the sum of the probabilities over all possible outcomes is 1.

2. Sigma-algebra的定义：

​ 假设$\Omega$是一个非空集合（通常称为样本空间），则一个 $\sigma$代数$\mathcal{F}$是该集合上的一个子集族，满足以下三个条件：

  1. 包含全集：$\Omega \in \mathcal{F}$。即整个样本空间是$\sigma$代数中的一个元素。
  2. 闭合于补集：如果$A \in \mathcal{F}$，则$A^c \in \mathcal{F}$，其中$A^c$表示事件$A$的补集。也就是说，$\sigma$代数中的每个事件的补集也必须在$\sigma$代数中。
  3. 闭合于可列并：如果$A_1, A_2, A_3, \dots \in \mathcal{F}$ 是$\sigma$代数中的一列事件，那么它们的并集$\bigcup_{n=1}^{\infty} A_n \in \mathcal{F}$也必须在$\sigma$代数中。这意味着$\sigma$代数对于可数的并集运算是封闭的。
解释：
  • 事件：在概率论中，事件可以看作是样本空间$\Omega$的子集。$\sigma$代数就是一组我们可以分配概率的事件集。
  • 闭合性：$\sigma$代数必须对补集操作和无限并集操作封闭，这样可以确保定义在这个代数上的测度（例如概率测度）有良好的性质，并且我们可以合理地处理复杂的事件。
示例：
  1. 最小的$\sigma$代数：最简单的情况是$\Omega = {1, 2, 3}$，最小的$\sigma$代数就是$\mathcal{F} = {\emptyset, \Omega}$，仅包含空集和全集。
  2. 幂集：样本空间$\Omega = {1, 2}$，它的幂集（即所有子集的集合）就是$\mathcal{F} = {\emptyset, {1}, {2}, {1, 2}}$。幂集是一个$\sigma$代数，因为它满足上述三条条件。
  3. Borel $\sigma$代数：在实数集合$\mathbb{R}$上，最常见的$\sigma$代数是Borel $\sigma$代数，它由所有的开区间$(a,b) \subseteq \mathbb{R}$ 生成。Borel $\sigma$代数包含了实数集的所有可以通过开区间构造出来的集合，也是定义概率测度和Lebesgue测度的重要工具。
重要性：
  • 定义概率空间：在概率论中，样本空间$\Omega$上的$\sigma$代数$\mathcal{F}$和概率测度$P$一起构成了一个概率空间$(\Omega, \mathcal{F}, P)$，这允许我们对事件集分配概率。
  • 可测性：$\sigma$代数是可测空间的基础，随机变量的定义依赖于$\sigma$代数。随机变量是可测函数，即它将样本空间$\Omega$映射到另一个空间的过程中，保持可测性结构。
